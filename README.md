# Mystery Manor Images

This repository contains image assets for the Mystery Manor project.

## How to Use

- **Direct image links:**  
  Images are accessible via GitHub Pages at:  
  ```
  https://drj-ctrl.github.io/Mystery_Manor_Images/<image-filename>
  ```
  Replace `<image-filename>` with the exact file name and extension (case-sensitive).

- **Example:**  
  [Blue_Flame_image.png](https://drj-ctrl.github.io/Mystery_Manor_Images/Blue_Flame_image.png)  
  ```
  https://drj-ctrl.github.io/Mystery_Manor_Images/Blue_Flame_image.png
  ```
