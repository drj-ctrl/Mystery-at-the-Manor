Mystery Manor Images

This is the image repository for your game assets.

  • To use an image, find the exact file name in the repo on GitHub.
  • Construct the image URL as:
    https://drj-ctrl.github.io/Mystery_Manor_Images/<your-image-file>
    (replace <your-image-file> with the exact name and extension)
  • If you want a gallery or image list here in the future, you can add it any time!
